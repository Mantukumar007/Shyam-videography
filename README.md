## Getting Started

### Prerequisites
- Node.js and npm (if using a JS-based site)
- Git
- (Optional) Static site generator CLI if applicable

### Installation
1. Clone the repo
   git clone https://github.com/Mantukumar007/Shyam-videography.git
2. Enter the project directory
   cd Shyam-videography
3. Install dependencies (if applicable)
   npm install

### Running Locally
- For a simple static site: open index.html in your browser
- For a Node-based site:
  npm run dev
Then open http://localhost:3000 (or the port your project uses).

## Deployment
- Build for production:
  npm run build
- Deploy to Netlify / Vercel / GitHub Pages or upload built files to your host.
- If using GitHub Pages: enable Pages in repo settings and point to the build/output folder.
